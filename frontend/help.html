User Guide

Here's how to navigate and use the system:

  • Go to Machines to see live machine statuses.
  • Use Faults to report and track machine issues.
  • Access Dashboard for a quick overview of all activities.

Frequently Asked Questions

  • Q: How do I report a machine fault?
    A: Go to the "Faults" page and click "Report New Fault".
  • Q: Can I track past repairs?
    A: Yes, visit the Dashboard or Machines page for repair history.
  • Q: Who do I contact if I’m locked out of my account?
    A: Please reach out to the support team via the Contact page.

API Documentation

Developers can integrate external systems with our platform using these basic API endpoints:

  • GET /api/machines – Retrieve all machines and statuses
  • POST /api/faults – Submit a new fault report
  • GET /api/repairs/:id – Get details of a specific repair

More detailed API documentation will be available soon.
